# 10 秒像素风即梦执行包

## 1. 先做这几步
1. 生成或导入 `ASSET-CHAR-A` 和 `ASSET-SCENE-A`。
2. 在 `CV-MASTER` 建立资产区和 `Z-S01` 到 `Z-S03`。
3. 按 `CV-OP-01` 到 `CV-OP-04` 完成构图与融合。
4. 导出 `IMG-S01` 到 `IMG-S03`。
5. 用对应导出执行 `VID-S01` 到 `VID-S03`，按 3s + 3s + 4s 剪成 10 秒。

## 2. 一句话设定
雨后像素森林里，小蘑菇帮助尾灯变暗的萤火虫重新发光。

## 3. 全局锚点
- 角色：矮圆小蘑菇，红色伞帽固定 3 个奶白圆点，浅米色身体，黑色豆豆眼；小圆萤火虫，深青绿色身体，透明短翅，暖黄色圆尾灯。
- 场景：雨后夜晚森林，苔藓、宽叶、圆露水、小花，蓝紫夜色和局部暖黄萤光。
- 风格：复古像素风，低分辨率游戏画面，清晰像素边缘，可爱治愈。
- 避免：写实昆虫、3D 玩具、厚涂插画、恐怖森林、文字、水印、角色变形。

| S02 | 3s | 小蘑菇托起一滴露水递过去 | 萤火虫慢慢靠近 | 图生视频 |

### 5.2 素材准备

#### IMG-REF / ASSET-CHAR-A + ASSET-CHAR-B + ASSET-SCENE-A
来源：生成；用户已有角色或场景图时直接导入并跳过对应生成。
素材提示词：
```text
可爱复古像素风角色和场景参考图，矮矮圆圆的小蘑菇，红色蘑菇伞帽固定 3 个奶白色圆点，浅米色身体，两颗黑色豆豆眼；旁边是一只小圆萤火虫，深青绿色身体，透明短翅，尾部圆形暖黄色发光点。雨后夜晚森林，苔藓、宽叶、圆露水、小花，蓝紫夜色，清晰像素边缘，温柔治愈。避免写实、3D、文字、水印。
```
完成检查：蘑菇帽子固定 3 个白点；萤火虫尾灯为暖黄色；森林元素和像素边缘清楚。

### 5.3 画布操作

#### CV-OP-01 建立资产区与镜头区域
画布/区域：`CV-MASTER / Z-ASSET、Z-S01-Z-S03`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`arrange`
完成检查：资产区与三个镜头区域互不覆盖；每个镜头预留 9:16 构图空间。
失败后改法：先只摆角色和场景轮廓，确认构图后再添加露水、小花等道具。

#### CV-OP-02 融合镜头 S01
画布/区域：`CV-MASTER / Z-S01`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`blend`
操作提示词：
```text
保留小蘑菇红色伞帽固定 3 个奶白圆点、浅米色身体和黑色豆豆眼。让小蘑菇自然站在雨后苔藓上并抬头看向叶尖，叶尖的深青绿色小萤火虫尾灯只有微弱暖黄光。统一蓝紫夜色、接触阴影和清晰像素边缘，中景平视，不增加新角色。
```
完成检查：角色比例不变；萤火虫位于视线方向；夜景可读但不发黑。
失败后改法：取消融合，只保留摆放构图并局部修复角色脚下接触阴影。
导出为：`IMG-S01`

#### CV-OP-03 局部修复镜头 S02
画布/区域：`CV-MASTER / Z-S02`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`inpaint`
操作提示词：
```text
只修改小蘑菇双手和露水区域：让小蘑菇轻轻托住一滴圆润露水，萤火虫从前方慢慢靠近。保留红色伞帽固定 3 个奶白圆点、角色比例、蓝紫雨夜森林和像素边缘，不生成复杂手指，不把露水变成玻璃球。
```
完成检查：露水轮廓清楚；手部简化；萤火虫与露水之间有可读距离。
失败后改法：保持小蘑菇静止托举，只移动萤火虫位置。
导出为：`IMG-S02`

#### CV-OP-04 统一镜头 S03 光影
画布/区域：`CV-MASTER / Z-S03`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`inpaint`
操作提示词：
```text
只调整萤火虫尾灯和周围受光区域：让暖黄色小光晕柔和照亮小蘑菇、小花、苔藓和宽叶，保留蓝紫夜色和角色轮廓。小蘑菇红色伞帽仍有固定 3 个奶白圆点，豆豆眼呈开心表情。光线不过曝，不变成霓虹效果。
```
完成检查：光源来自萤火虫尾灯；小蘑菇轮廓完整；暖黄与蓝紫对比稳定。
失败后改法：缩小修复选区，只增强尾灯和角色脸部附近的局部受光。
导出为：`IMG-S03`

## 7. 失败修正
- 像素风变写实：把提示词前半段加强为“复古像素风、低分辨率游戏画面、清晰像素边缘”。
- 角色漂移：每条提示词重复“红色伞帽固定 3 个奶白圆点”和“暖黄色圆尾灯”。
- 光效过曝：把“发光”改成“柔和暖黄色小光晕，照亮局部，不覆盖轮廓”。

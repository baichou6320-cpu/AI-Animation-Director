# 10 秒像素风即梦画布执行包

## 1. 先做这几步
1. 生成或导入小蘑菇、萤火虫和雨夜森林素材。
2. 按 `S01 -> S02 -> S03` 完成关键帧和视频，不必先做完所有首帧。
3. 完成一个编号后可回复“S01 完成，继续”，后续只返回下一步。

## 2. 项目锚点与镜头表
设定：雨后像素森林里，小蘑菇帮助尾灯变暗的萤火虫重新发光。

- 角色：红帽小蘑菇固定 3 个奶白圆点；深青绿色小萤火虫固定暖黄色圆尾灯。
- 场景：雨后夜晚森林，苔藓、宽叶、圆露水和小花。
- 风格：复古像素风，清晰像素边缘，蓝紫夜色与局部暖黄光。
- 避免：写实昆虫、3D 玩具、厚涂、恐怖气氛、文字、水印、角色变形。

| 镜头 | 时长 | 画面 | 动作 | 即梦方式 |
| --- | --- | --- | --- | --- |
| S01 | 3s | 小蘑菇发现微弱萤火虫 | 抬头，尾灯轻闪 | 画布关键帧 + 图生视频 |
| S02 | 3s | 小蘑菇递出露水 | 萤火虫缓慢靠近 | 画布关键帧 + 图生视频 |
| S03 | 4s | 暖光照亮森林 | 尾灯变亮，小蘑菇眨眼 | 画布关键帧 + 图生视频 |

## 3. 素材准备
画布地图：`CV-MASTER: Z-ASSET + Z-S01...Z-S03`

### IMG-REF / ASSET-CHAR-A + ASSET-CHAR-B + ASSET-SCENE-A
来源：无已有素材时生成；已有角色或场景图时直接导入。
素材提示词：
```text
可爱复古像素风角色和场景参考图。矮小圆润的小蘑菇，红色伞帽固定 3 个奶白圆点，浅米色身体，黑色豆豆眼；小圆萤火虫，深青绿色身体，透明短翅，尾部暖黄色圆形发光点。雨后夜晚森林，苔藓、宽叶、圆露水和小花，蓝紫夜色，清晰像素边缘，温柔治愈。避免写实、3D、文字、水印。
```
完成检查：
- 蘑菇帽子始终是 3 个白点。
- 萤火虫尾灯为暖黄色，场景保持像素边缘。

## 4. 逐镜头执行卡

### S01：`Z-S01 -> IMG-S01 -> VID-S01`

#### 画布关键帧
操作编号：`CV-OP-01`
画布/区域：`CV-MASTER / Z-S01`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`blend`
操作提示词：
```text
保留小蘑菇红色伞帽固定 3 个奶白圆点、浅米色身体和豆豆眼。让它站在雨后苔藓上，抬头看向叶尖的深青绿色小萤火虫；萤火虫尾灯只有微弱暖黄光。统一蓝紫夜色、接触阴影和清晰像素边缘，中景平视，不增加新角色。
```
完成检查：
- 视线方向清楚，夜景可读。
- 角色比例和帽子白点不变。
失败后改法：取消融合，只摆放角色和场景，再局部补脚下阴影。
导出为：`IMG-S01`

#### 视频生成
任务编号：`VID-S01`
推荐方式：图生视频
使用图片：`IMG-S01`
复制提示词：
```text
从这张像素风首帧开始。小蘑菇轻轻抬头看向叶尖，萤火虫暖黄色尾灯微弱闪烁。镜头固定或极慢推近，雨后森林背景稳定，角色造型不变，保持复古像素风和清晰像素边缘。
```
完成检查：
- 只发生抬头和尾灯轻闪。
- 背景、帽子白点和角色比例稳定。
失败后改法：固定镜头，只保留尾灯轻微闪烁。

### S02：`Z-S02 -> IMG-S02 -> VID-S02`

#### 画布关键帧
操作编号：`CV-OP-02`
画布/区域：`CV-MASTER / Z-S02`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`inpaint`
操作提示词：
```text
只修改小蘑菇双手和露水区域：让小蘑菇轻轻托住一滴圆润露水，萤火虫从前方缓慢靠近。保留红色伞帽固定 3 个奶白圆点、角色比例、蓝紫雨夜森林和像素边缘，不生成复杂手指，不把露水变成玻璃球。
```
完成检查：
- 露水轮廓清楚，手部保持极简。
- 萤火虫和露水之间有可读距离。
失败后改法：让小蘑菇静止托举，只调整萤火虫位置。
导出为：`IMG-S02`

#### 视频生成
任务编号：`VID-S02`
推荐方式：图生视频
使用图片：`IMG-S02`
复制提示词：
```text
从这张像素风首帧开始。小蘑菇保持托举姿势，慢慢把圆露水递向萤火虫，萤火虫只轻轻靠近。镜头固定，露水保持圆润，背景不切换，角色造型和帽子 3 个白点不变。
```
完成检查：
- 露水不变形，不出现复杂手指。
- 只有递出和轻轻靠近两个关联动作。
失败后改法：小蘑菇完全静止，只让萤火虫靠近。

### S03：`Z-S03 -> IMG-S03 -> VID-S03`

#### 画布关键帧
操作编号：`CV-OP-03`
画布/区域：`CV-MASTER / Z-S03`
输入素材：`ASSET-CHAR-A`、`ASSET-CHAR-B`、`ASSET-SCENE-A`
操作类型：`inpaint`
操作提示词：
```text
只调整萤火虫尾灯和周围受光区域：暖黄色小光晕柔和照亮小蘑菇、小花、苔藓和宽叶，保留蓝紫夜色和角色轮廓。小蘑菇红色伞帽仍有固定 3 个奶白圆点，豆豆眼呈开心表情。光线不过曝，不变成霓虹效果。
```
完成检查：
- 光源明确来自萤火虫尾灯。
- 暖黄与蓝紫对比稳定，角色轮廓完整。
失败后改法：缩小修复选区，只增强尾灯和脸部附近受光。
导出为：`IMG-S03`

#### 视频生成
任务编号：`VID-S03`
推荐方式：图生视频
使用图片：`IMG-S03`
复制提示词：
```text
从这张像素风首帧开始。萤火虫尾灯慢慢变亮，柔和暖黄光照亮小蘑菇、小花和苔藓，小蘑菇轻轻眨眼。镜头轻微推近，蓝紫夜色稳定，光晕不过曝，角色不变形。
```
完成检查：
- 光线缓慢变亮且不覆盖角色轮廓。
- 小蘑菇只眨眼，不增加额外动作。
失败后改法：固定镜头，只做尾灯变亮和眨眼。

## 5. 失败修正
- 像素风变写实：强化“复古像素风、低分辨率游戏画面、清晰像素边缘”。
- 角色漂移：每条提示词重复“红帽固定 3 个白点”和“暖黄色圆尾灯”。
- 光效过曝：改为“柔和小光晕，只照亮局部，不覆盖轮廓”。

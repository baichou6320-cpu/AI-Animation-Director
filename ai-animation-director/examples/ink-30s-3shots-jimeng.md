# 30 秒国风水墨即梦画布执行包

## 1. 先做这几步
1. 生成或导入旅人、江岸和纸灯素材。
2. 按 `S01 -> S02 -> S03` 完成关键帧和视频，优先使用扩图与局部修复。
3. 完成一个编号后可回复“S01 完成，继续”，后续只返回下一步。

## 2. 项目锚点与镜头表
设定：水墨江岸边，旅人放下纸灯与友人告别，纸灯顺水远去。

- 角色：青灰长衫旅人，简洁发髻；远处友人只保留淡墨剪影。
- 场景：清晨江岸、芦苇、石阶、纸灯、薄雾和水面倒影。
- 风格：宣纸纹理、墨色扩散、大片留白，淡青灰与一点暖灯色。
- 避免：浓艳色彩、写实摄影、复杂动作、快速镜头、文字、水印、3D。

| 镜头 | 时长 | 画面 | 动作 | 即梦方式 |
| --- | --- | --- | --- | --- |
| S01 | 8s | 旅人站在江岸，友人剪影在雾里 | 旅人轻轻低头 | 画布关键帧 + 图生视频 |
| S02 | 10s | 旅人把纸灯放入水中 | 纸灯缓慢漂开 | 画布关键帧 + 图生视频 |
| S03 | 12s | 纸灯远去，旅人回望 | 薄雾淡开 | 画布关键帧 + 图生视频 |

## 3. 素材准备
画布地图：`CV-MASTER: Z-ASSET + Z-S01...Z-S03`

### IMG-REF / ASSET-CHAR-A + ASSET-SCENE-A + ASSET-PROP-A
来源：生成；已有水墨角色或场景素材时优先导入。
素材提示词：
```text
国风水墨动画参考图，宣纸纹理、淡墨线条和墨色扩散。清晨江岸，芦苇、石阶、小舟、薄雾、水面倒影，大片留白。一位青灰长衫旅人，简洁发髻；远处友人是淡墨剪影。画面淡青灰为主，纸灯一点温暖金色，不写实，不 3D，无文字水印。
```
完成检查：
- 旅人造型简洁，纸灯是唯一暖色。
- 场景留白充足，没有复杂建筑。

## 4. 逐镜头执行卡

### S01：`Z-S01 -> IMG-S01 -> VID-S01`

#### 画布关键帧
操作编号：`CV-OP-01`
画布/区域：`CV-MASTER / Z-S01`
输入素材：`ASSET-CHAR-A`、`ASSET-SCENE-A`
操作类型：`expand`
操作提示词：
```text
向画面右侧自然扩展清晨江面、薄雾和宣纸留白，延续淡青灰墨色、轻微水面倒影和横向构图。保留青灰长衫旅人、石阶和远处友人剪影不变，不增加建筑、文字或浓重景物。
```
完成检查：
- 扩展区域无接缝，人物比例不变。
- 留白仍是画面主体。
失败后改法：减少扩展范围，只补江面和薄雾。
导出为：`IMG-S01`

#### 视频生成
任务编号：`VID-S01`
推荐方式：图生视频
使用图片：`IMG-S01`
复制提示词：
```text
从这张水墨首帧开始。旅人轻轻低头，远处友人剪影保持静止，薄雾缓慢流动。镜头固定，墨色自然晕开，留白稳定，动作极慢，保持国风水墨和宣纸纹理。
```
完成检查：
- 只有低头和薄雾缓慢流动。
- 留白、人物比例和淡墨风格稳定。
失败后改法：去掉人物动作，只保留薄雾轻微流动。

### S02：`Z-S02 -> IMG-S02 -> VID-S02`

#### 画布关键帧
操作编号：`CV-OP-02`
画布/区域：`CV-MASTER / Z-S02`
输入素材：`ASSET-CHAR-A`、`ASSET-SCENE-A`、`ASSET-PROP-A`
操作类型：`inpaint`
操作提示词：
```text
只修改纸灯、手部和接触水面的区域：让旅人以简化水墨动作把一盏暖金色小纸灯放入水中，水面只有两三圈淡墨水纹。保留青灰长衫、石阶、薄雾、宣纸纹理和大片留白，不生成写实手指或复杂波浪。
```
完成检查：
- 纸灯完整，手部不抢眼。
- 水纹克制，暖色不过亮。
失败后改法：去掉人物手部，只保留已漂离石阶的纸灯。
导出为：`IMG-S02`

#### 视频生成
任务编号：`VID-S02`
推荐方式：图生视频
使用图片：`IMG-S02`
复制提示词：
```text
从这张水墨首帧开始。旅人缓慢把纸灯放入水中，纸灯轻轻离开手边，水纹很轻。镜头固定，动作克制，墨色淡雅，纸灯暖光不过亮，背景不切换。
```
完成检查：
- 纸灯缓慢移动，水纹不过度扩散。
- 人物和背景保持稳定。
失败后改法：让旅人静止，只让纸灯缓慢漂开。

### S03：`Z-S03 -> IMG-S03 -> VID-S03`

#### 画布关键帧
操作编号：`CV-OP-03`
画布/区域：`CV-MASTER / Z-S03`
输入素材：`ASSET-CHAR-A`、`ASSET-SCENE-A`、`ASSET-PROP-A`
操作类型：`inpaint`
操作提示词：
```text
只调整远处薄雾、友人剪影和纸灯周围：让友人剪影自然融入淡墨薄雾，暖金色纸灯顺水远去并保持为唯一亮点。保留旅人回望姿态、大片留白、宣纸纹理和淡青灰横向构图，不增加新景物。
```
完成检查：
- 友人剪影若隐若现，纸灯仍可读。
- 画面没有被雾气填满。
失败后改法：固定人物和背景，只降低友人剪影对比度。
导出为：`IMG-S03`

#### 视频生成
任务编号：`VID-S03`
推荐方式：图生视频
使用图片：`IMG-S03`
复制提示词：
```text
从这张水墨首帧开始。纸灯顺水慢慢远去，镜头极慢横向移动，薄雾淡开，旅人安静回望。保持大片留白、宣纸纹理和淡青灰墨色，动作很少，节奏舒缓。
```
完成检查：
- 横移极慢，纸灯始终可见。
- 留白和宣纸纹理保持稳定。
失败后改法：固定镜头，只让纸灯慢慢漂远。

## 5. 失败修正
- 水墨变写实：加强“宣纸纹理、淡墨线条、墨色扩散、大片留白”。
- 动作过复杂：删除手部细节，只保留“纸灯缓慢漂开”。
- 画面太满：加入“大片留白、远景、低元素密度、克制构图”。

## 官方确认能力

即梦中国官网将“智能画布”描述为一站式画布，确认包含：

- 多图 AI 融合与 AI 拼图生成。
- 同一画布上的多元素拼接。
- 多图层编辑。
- 局部重绘。
- 一键扩图。
- 图像消除。
- 抠图。
- 用于统一画面风格与组合多个视觉元素。

官方页面：

- https://jimeng.jianying.com/

Dreamina 官方页面进一步说明了同产品体系下的画布与图像编辑工作流：

- 上传前景与背景素材后进行融合。
- 通过提示词描述融合后的风格或环境。
- 调整素材位置后再融合。
- 扩展现有图像并维持原有构图与美术特征。
- 对指定区域进行交互式编辑，尽量保留周围纹理和光线。
- 完成后导出选中图像或整张画布。
- 可在完成构图后使用图像放大功能。

官方页面：

- https://dreamina.capcut.com/tools/ai-blender
- https://dreamina.capcut.com/tools/ai-photo-editor

## 本 Skill 的稳定操作语义

Skill 使用下列抽象操作，不绑定网页按钮名称：

| 操作 | 用途 | 是否通常需要提示词 |
| --- | --- | --- |
| `generate/import` | 生成或导入角色、场景、道具、产品和参考图 | 生成时需要 |
| `arrange` | 调整位置、大小、前后层级和构图 | 否 |
| `cutout` | 从原背景分离角色、产品或道具 | 否 |
| `blend` | 统一素材边缘、空间、光影和风格 | 是 |
| `inpaint` | 修复选区内的局部内容 | 是 |
| `expand` | 扩展画面、改变画幅或增加环境空间 | 是 |
| `remove` | 清除多余元素或伪影 | 视情况 |
| `upscale` | 在结构正确后提高可用尺寸与细节 | 否 |
| `export` | 导出选中图像或画布成果 | 否 |

## 动画生产中的推荐用途

### 资产母版

将角色正侧面、场景、道具、产品和色彩参考集中管理，作为所有镜头的视觉来源。

### 分镜构图

把角色和场景素材摆到目标景别与机位，先确认构图，再决定是否需要 AI 融合。

### 一致性修复

局部重绘只处理明确选区，例如：

- 表情或眼睛。
- 服装和标志物。
- 手部与道具接触。
- 光源方向。
- 多余人物或错误物件。

### 画幅适配

使用扩图为横屏、竖屏或更宽景别补足环境空间。扩图提示词应强调延续原场景、光线、材质和透视。

### 视频首帧交付

构图与一致性通过后导出 `IMG-Sxx`，再交给对应 `VID-Sxx` 做图生视频。

## 不稳定或未验证信息

以下内容可能因地区、账号、版本、会员或界面更新而变化，Skill 不应写死：

- 具体按钮名称与位置。
- 可上传素材数量。
- 画布尺寸或图层数量上限。
- 融合强度范围。
- 实时模型名称。
- 分辨率和导出格式选项。
- 免费额度、积分消耗和会员权益。
- 批量处理数量。
- 是否提供公开画布 API。

需要这些信息时，提示用户以当前账号界面和官方文档为准。

## 安全与边界

- 第一版仅生成人工执行方案，不自动操作即梦网页。
- 不要求用户提供 cookie、session token 或账号密码。
- 不把未经确认的界面行为描述成稳定 API。
- 不把画布当作视频时间线；画布负责视觉资产和关键帧，视频模块负责运动。
- 不用多次高强度融合代替角色与场景圣经。

## 快速决策表

| 问题 | 首选操作 |
| --- | --- |
| 角色和背景像贴上去的 | `blend` |
| 角色外观局部漂移 | `inpaint` |
| 横图改竖图或景别太紧 | `expand` |
| 多余人物、文字或伪影 | `remove` |
| 需要重复使用角色 | `cutout` 后放入资产区 |
| 构图尚未确定 | `arrange`，不要先融合 |
| 图像结构错误 | 返回生图或局部重绘，不要直接 `upscale` |
| 画面已通过检查 | `upscale` 后 `export` |
